# 자바 IO, NIO

## 자바 IO와 NIO의 차이

| 구분                 | IO                 | NIO                            |
| -------------------- | ------------------ | ------------------------------ |
| 입출력 방식          | 스트림 방식        | 채널 방식                      |
| 버퍼 방식            | 넌버퍼             | 버퍼                           |
| 비동기 방식          | 지원 안 함         | 지원                           |
| 블로킹/넌블로킹 방식 | 블로킹 방식만 지원 | 블로킹/넌블로킹 방식 모두 지원 |
